# japan-colors-syntax theme

An atom.io color theme based on
[The Traditional Colors of Japan](http://nipponcolors.com)

![Screenshot](Screenshot.png)

The colors included

@fujimurasaki        : #8A6BBE; // 藤紫
@shironeri           : #FCFAF2; // 白練
@shironezumi         : #BDC0BA; // 白鼠
